# newPopup 弹出层

## 介紹
弹出层容器，用于展示弹窗、信息提示等内容，支持多个弹出层叠加展示

底层弹框组件，主要用于基于此组件实现上层组件的封装。只提供基础功能：指定类型、是否需要背景层、显示内容、居中（定位）

内置弹层类组件基于此组件实现，包括：[Toast](Toast)、[Modal](Modal)、[Picker](Picker)

## 使用
<Playground>
  {()=>{
      const Temp = () => {
        const [isShow, setIsShow] = React.useState(false);
        return (
            <div>
              <Button.Group>
                <Button.Self color="blue" onClick={()=>{
                    setIsShow(true);
                  }} primary>确认</Button.Self>
              </Button.Group>
              <Popup isShow={isShow} onClose={() => {
                  setIsShow(false);
                }}>
                <div style={{ padding: '10px', backgroundColor: '#fff'}}>你好啊，小朋友</div>
              </Popup>
            </div>
        );
      };

      return <Temp/>;
    }}
</Playground>



## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| isShow | 是否显示 | Boolean | true/false | `false` |
| enableMask | 是否显示蒙层 | Boolean | true/false | `false` |
| children | 子元素 | Object/String/Number | - | - |
| className | class | String | - | - |




## 事件
| 事件名称 | 说明 | 返回参数 |
|  :---  |  :--- |  :----  |
| onClose | 关闭事件 | - | 




> 注释： 
* 使用组件时，指定类型 type，这样方便根据类型做 class 控制。例如
`<Popup type="my-popup"></Popup>`, 会在根元素上添加 `jqb-my-popup` 的 class
* isLock 参数，当弹出弹层后希望能锁住屏幕（如果可以下拉禁止下拉）
