## 参数
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :--- | :--- |
| close | 是否需要关闭按钮 | Boolean | - | false | 
| marquee | 内容是否滚动(内容只能单行) | Boolean | - | false |
| speed | 内容滚动速度(默认速度100px/秒) | Number | - | 100 |
| showMore | 文字“显示更多”显示 | Boolean | - | false |
| icon | 内容前的icon图标 | String | 参考`icon`图标 | - |
| className | noticebar样式 | String | - | - |
| children | noticebar内容 | reactnode | - | - |
