# Tag 标签 

## 介绍
用于展示1个或多个文字标签，可点击切换选中、不选中的状态。

## 使用
<Playground>
  <Tag>标签</Tag>
  <Tag size="normal">标签</Tag>
  <Tag color="#1496ff">标签1</Tag>
  <Tag color="#ea2e49">标签2</Tag>
</Playground>


## Api
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| color | 自定义颜色 | String | 任意颜色色号 | #ddd
| size | 大小尺寸 | String  | normal, small |  small
