## 基础使用
<Playground>
    {()=>{
      const Temp = () => {
        const [status, setIsStatus] = React.useState(false);

        return (
            <div>
                <Switch
                    status={status}
                    onChange={(e, isOn) => {
                        {/* console.log('e',e) */}
                        {/* console.log('isOn', isOn) */}

                    }}
                />
            </div>
        );
      };

      return <Temp/>;
    }}

</Playground>

## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| className | 自定义class | String | - | - |
| status | 开关选中状态 | Boolean | `true` `false` | - |
| loading | 是否为加载状态 | Boolean | `true` `false` | - |
| disabled | 是否为禁用状态 | Boolean | `true` `false` | - |
| activeColor | 打开时的背景色 | String | - | `#4bd764` |


## 事件
| 事件名称 | 说明 | 参数 |
| :--- | :--- | :--- |
| onChange | 点击后的回调 | e: 事件, isOn: 开关选中状态 |
