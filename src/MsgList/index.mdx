## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| msgDate | 左上角日期 | String | - | - |
| msgCheck | 查看详情 | boolean | `true` `false` | `false` |
| isSign | 标识 | boolean | `true` `false` | `false` |
| isRead | 已读 | boolean | `true` `false` | `false` |
| msgContent | 消息内容 | string | `true` `false` | `false` |
| contentLength | 内容最长字符数 | number | - | 100 |

> contentLength 默认50个中文字符，中文2个字符,英文1个字符

## 事件

| 事件名称 | 说明 | 参数 |
| :--- | :--- | :--- |
| onCheckClick | 点击查看消息的回调 | - |
