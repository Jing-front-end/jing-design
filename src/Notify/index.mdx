# Notify 消息提示


## 使用（一）
<Playground>
  <div>
    <Button.Group>
      <Button.Self color="blue" onClick={()=>{
          Notify.success('消息提示-成功');
        }} primary>确认</Button.Self>
      <Button.Self color="orange" onClick={()=>{
        Notify.warning('消息提示-警告');
      }} primary>确认</Button.Self>
            <Button.Self color="orange" onClick={()=>{
        Notify.warn('消息提示-警告');
      }} primary>确认</Button.Self>
      <Button.Self color="red" onClick={()=>{
        Notify.error('消息提示-错误');
      }} primary>确认</Button.Self>
      <Button.Self color="yellow" onClick={()=>{
        Notify.info('消息提示-正常', 10);
      }} primary>确认</Button.Self>
    </Button.Group>
  </div>
</Playground>


## 静态方法
- Notify.success(text: node, duration?: number)
- Notify.warning(text: node, duration?: number)
- Notify.warn(text: node, duration?: number)
- Notify.error(text: node, duration?: number)
- Notify.info(text: node, duration?: number)

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| text | 提示内容 |  string |  - | - |
| duration | 持续时间 | number | - | 3 |
