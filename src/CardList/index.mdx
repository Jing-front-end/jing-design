## props

| 参数 |   说明   |  类型  | 默认值 |
| :--- | :------- | :----- | :----- |
| list | 银行列表 | array  |        |
| type | 种类     | string | small  |



### list

|          参数          |       说明       |   类型   |
| :--------------------- | :--------------- | :------- |
| link                   | 链接地址         | string   |
| cardPkid               | 银行卡id         | string   |
| bankAccountEndFour     | 后四位           | string   |
| bankLimitDay           | 日限额           | string   |
| bankName               | 银行名称         | string   |
| convertedBankLimit     | 单笔首次限额     | string   |
| convertedNextBankLimit | 单笔后续限额     | string   |
| convertedBankLimitDay  | 单日限额         | string   |
| bankNo                 | 银行编号         | string   |
| onBindTipsClick        | 绑定注册事项点击 | function |


### type

|  参数   |   说明   |  类型  |
| :------ | :------- | :----- |
| small   | 小卡     | string |
| large   | 大卡     | string |
| primary | 突出底色 | string |
