# TabBar 标签栏

暂不支持 纵向排列

## TabBar API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| className | 自定义样式名 | String | - | - |
| children | 排列的TabBarItem | ReactNode | - | - |
| value | 传给Item的索引值 | Number | - | - |
| indicator | 是否显示下划线 | Boolean | `true` `false` | `false` |
| sticky | 是否显示吸附效果 | Boolean | `true` `false` | `false` |


## TabBar 事件

| 事件名称 | 说明 | 参数 |
| :--- | :--- | :--- |
| onChange | 传给Item的点击事件 | e: 点击事件event, value: 选中的索引 |


## Item API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| icon | 图标 | String | `Icon` 中可选 | - |
| children | Item中的文本 | Node | - | - |
| value | 当前Item的索引值 | Number | - | - |
| selected | 判断当前值是否选择 | Boolean | - | - |
| tabBarItemStyle | Item 的样式 | object & array | - | - |



## Item 事件

| 事件名称 | 说明 | 参数 |
| :--- | :--- | :--- |
| onChange | 点击Item后的回调 | e: 点击事件event, value: 选中的索引 |
