# 加载

图标大小默认为 `24px`, 可以通过`size`属性配置 

## 使用
<Playground>
  <Loading />

  <Loading size="40px"/>
</Playground>

## Tag 参数
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| size | 大小尺寸 | String  | - |  24px
