# 模态框

所有模态框如 modal， 
此模态框是基础模态框，一种为常见的modal，通过参数配置，
另一种更自由化，通过组件内写组件的形式来谋划
ps：没有复写alert，alertSelect，在鄙人看来，这些组件可以用过“自由化”组件来改造，应写在业务组件中

目前只支持一层弹层，如项目需要再写多层

## Modal 一般用法
### 参数
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
|   title  |      标题     | String |  - |  -
| content  |      内容     | String |  - |  -
|cancelText|  取消按钮文本  | String |  - |  取消
|confirmText|  确认按钮文本  | String |  - |  确认
|closeOnClickOverlay| 点击浮层的时候关闭模态框 | Boolean | - | true
| Backdrop | 是否显示背景 | Boolean | `true/false` | true |
| size | 模态框尺寸 | String | `sm/md/lg` | md |
| maxHigher | 是否计算模态框最大高度 | Boolean | `true/false` | true | 


## Modal 弹出框
### 事件
| 事件名称 | 说明 | 返回参数 |
| :--- | :--- | :---- |
| onClose | 关闭事件 | 
| onCancel | 取消事件 | 
| onConfirm | 确认事件 |
