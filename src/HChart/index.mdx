## HChart 图表

## 说明
- 图表功能
- 封装 hightchart, 具体文档看文档 hightchart[https://api.highcharts.com/highcharts/],中文文档[https://api.highcharts.com.cn]


## 使用
<Playground>
  <HChart
    className="test1"
    option={
      {
        xAxis: {
          categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec']
        },
        series: [{
          data: [29.9, 71.5, 106.4, 129.2, 144.0, 176.0, 135.6, 148.5, 216.4, 194.1, 295.6, 454.4]
        }]
      }
    }
  />
</Playground>

## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :--- | :--- |
| className | class | String | - | - |
| option | hightchart配置 | String | - | - |
