## API DetailList.Header
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| title | 标题 | String | - | - |


## API DetailList.Item
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| leftText | Item中左边的文字 | String | - | - |
| rightChildren | Item中右边的Node | Node | - | - |

## DetailList.Line
展示分割线

## DetailList.Body
包裹 DetailList.Item
