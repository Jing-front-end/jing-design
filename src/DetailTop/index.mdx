## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| bigText | 第一行大字体文本 | String | - | - |
| text | 第二行文本文字 | String | - | - |
| desc | 第三行描述文字 | String | - | - |
