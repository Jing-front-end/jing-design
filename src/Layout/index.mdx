## Props 配置
### Conatiner
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| type | 按钮类型 | String | button/submit | button



### Row
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| type | 按钮类型 | String | button/submit | button


### Col
| 参数 |     说明    | 类型 | 可选值 | 默认值 |
| :--- | :---       | :---- | :--- | :---- |
| as |   元素类型   | String | button/submit | div
| lg | >=992px尺寸 | Object  
| md | >=768px尺寸 |
| sm | >=576px尺寸 |
| xl | >=1200px尺寸 |
| xs | <576px尺寸 |

lg,md,sm,xl,xs 都有三
Boolean 不填的情况 true auto
填 number 按照大小布局，共12分，

> 注1：由于微信 button 的部分样式属性和 AtButton 有冲突，所以暂未被 AtButton 支持。支持的属性以上表为准

order: number 排序。数字越小，越往前排，只和有order 的做对比

offset: number 偏移 按照数字在row中偏移，只做往左偏移
