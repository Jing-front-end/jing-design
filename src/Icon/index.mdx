# Web

## Icon

### props
| 参数      | 说明                                      | 类型         | 默认值 |
|-----------|-------------------------------------------|--------------|--------|
| icon | 图标 | string |  |
| type | 类型 | string | normal |
| size | 图标 | string | normal |
| animate | 动画 | string | shake |
| hasColor | 图标 | bool | true |


### type
| 参数      | 说明                                      | 类型         |
|-----------|-------------------------------------------|--------------|
| normal | 常规图标 | string |  |
| feature | 结果图标 | string |  |


### size
| 参数      | 说明                                      | 类型         |
|-----------|-------------------------------------------|--------------|
| normal | 正常 | string |  |
| small | 小号 | string |  |
| mini | 迷你 | string |  |


### animate
| 参数      | 说明                                      | 类型         |
|-----------|-------------------------------------------|--------------|
| shake | 抖动效果 | string |  |


### icon
| 参数      | 说明                                      | 类型         |
|-----------|-------------------------------------------|--------------|
| xxxxxx | 太多的懒得写 | string |  |
