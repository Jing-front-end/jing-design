# Checkout 多选框


## 使用

## API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| className | checkbox的样式 | String | - | - |
| style | checkbox的style | String | - | - |
| data | 数据源,即Item的数据源 | Object | - | - |
| selected | 选中的数据 | String | - | - |

## 事件
| 事件名称 | 说明 | 参数 |
|  :---   | :--- | :--- |
| onChange | 点击后回调函数 | - |


## Item API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :--- | :---- | :--- |
| className | item的样式 | String | - | - |
| style | item的style | String | - | - |
| value | 当前Item是否被选中 | Boolean | false | - |
| disabled | 是否能点击 | Boolean | false | - |
