## Popup.Alert

<Playground>
  {()=>{
    const Temp = () => {
      const [isShow, setIsShow] = React.useState(false);

      return (
          <div>
            <Button.Group>
            <Button.Self color="blue" onClick={()=>{
                setIsShow(true);
              }} primary>确认</Button.Self>
            </Button.Group>
            <Popup.Group show={isShow} onClose={()=>{
                setIsShow(false);
            }}>
                <Popup.Alert
                    title='这个我必须要告诉你！'
                    onSubmit = {()=>{
                        setIsShow(false);
                    }}
                />
            </Popup.Group>
          </div>
      );
    };

    return <Temp/>;
  }}
</Playground>

|     参数      |     说明     |   类型   |
| :------------ | :----------- | :------- |
| title         | 提示内容     | string   |
| btnSubmitText | 确认按钮文字 | string   |
| onSubmit      | 点击事件     | function |
